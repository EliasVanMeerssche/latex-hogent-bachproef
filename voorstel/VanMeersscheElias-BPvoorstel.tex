\begin{abstract}
Tijdens deze bachelorproef wordt er een kernel module ontworpen en geïmplementeerd. Deze zal gebruikt worden tijdens een praktische oefening in het vak Cybersecurity. Het verborgen bestand of proces zal door de studenten gevonden moeten worden tijdens de oefening.
Er wordt onderzocht wat de beveiligingsmechanismen in het Linux besturingssysteem zijn, en welke methodes er bestaan om deze te omzeilen. Verschillende soorten kernel models worden bestudeerd en in een proof on concept wordt er vervolgens een kernel module  ontworpen die kan via een rootkit ingeladen kan worden en verstopt kan worden in het systeem.  

\end{abstract}
